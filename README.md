# immfly

# install

in root folder, create an .env with this structure
```
PRETTY_LOGS=1
APP_NAME=immfly
PORT=3028
TZ=Europe/Madrid
MAIN_DOMAIN=localhost
NODE_ENV=prod
SIMPLE_ARRAY="[1,2,3,4]"
```

# start

```
docker compose up
```

# postman

There's a postman collection in [postman/immfly.json](postman/immfly.json) for testing purposes
